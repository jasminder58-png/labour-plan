Why do I need to reconnect after closing?
Browsers enforce a security rule: a website can remember which folder you chose, but cannot silently re-read it without you clicking a button first. When the folder is already stored, clicking Reconnect (one click) restores access instantly — no need to navigate to the folder again.
🔗 SharePoint — No Azure Needed
Recommended: OneDrive Sync (2-minute setup, works forever)
1. In SharePoint, open your document library
2. Click Sync in the top toolbar — OneDrive syncs the folder to your PC automatically
3. In File Explorer, navigate to OneDrive – [Org] › Sites › [Your Library]
4. Point this dashboard at that local folder

The folder is now live — whenever someone updates a file in SharePoint, OneDrive syncs it to your PC within seconds. The 15-min auto-refresh picks it up automatically. No Azure, no login, no API keys.
